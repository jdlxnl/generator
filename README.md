###### Package

Add the package by loading it through composer.

```json

{
    "require": {
        "jdlx/jdlx": "*"
    },
    
    "require-dev" : {
        "jdlx/generator": "*"
    },
    
    "repositories": [
        {
            "type": "path",
            "url": "./jdlx/Foundation"
        },
        {
            "type": "path",
            "url": "./jdlx/Generator"
        }
    ]
}
```
